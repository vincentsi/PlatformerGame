# Référence des Plateformes - Zone 1 Level 2

Ce fichier liste toutes les plateformes avec leur index pour faciliter les modifications.

## Comment utiliser
- Chaque plateforme a un **index** (numéro dans le tableau)
- Dans le JSON, `platforms[0]` = première plateforme, `platforms[1]` = deuxième, etc.
- Les coordonnées sont en pixels (x, y) avec width et height

## Astuce pour modifier

Pour modifier une plateforme, trouve son index dans ce fichier, puis dans le JSON :
- `platforms[29]` = plateforme à l'index 29 (x: 3100, y: 200)
- Les indices commencent à 0, donc la première plateforme est `platforms[0]`

**Exemple** : Pour modifier la plateforme à x=1700, y=300 :
- C'est l'index 18 dans le tableau "ZONE MILIEU"
- Dans le JSON, c'est `platforms[18]` (ligne 29 du fichier JSON)
